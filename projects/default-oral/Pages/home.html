Skeleton Oral Corpus

This is an empty site to help set-up a new TEITOK corpus meant for oral data. This oral corpus provides a number of orally-oriented settings, that can be modified if necessary:

  • The CSS style provides a speaker-turn oriented view, with the @who of each utterance in front of each transcribed utterance
  • The CQP corpus exports start and end times on utterances, and displays the utterance context by default - this allow the sound of each utterance to be played directly from the search result list.
  • The teiHeader edit is set up for a URL to the audio file, as well as speaker-oriented data
  • File upload explicitly allows for uploading WAV files
