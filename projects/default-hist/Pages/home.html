Skeleton Historic Corpus

This is an empty site to help set-up a new TEITOK corpus meant for historic corpora. This historic corpus provides a number of manuscript-oriented settings, that can be modified if necessary:

  • Each word is provided with a normalised form (modernised, regularised), a diplomatic form, and an expanded form
  • The metadata are set up for a description of the manuscript, as well as author-oriented data
  • File upload explicitly allows for uploading Facsimile images

As the owner of this project you may wish to at least do the following things, and do the first two as soon as possible:

  • Login as nobody@nowhere.com / defaultpassword - which you will be asked to change
  • Run additional checks on your configuration in config check.
  • Replace the Smarty template and CSS style files to reflect your own look-and-feel (colours, logos, etc.).
  • Refined your settings.
  • Replace this page by a page explaining your project.

More information about how to set-up TEITOK can be found in the help pages: http://www.teitok.org/index.php?action=help
