Skeleton Learner Corpus

This is an empty site to help set-up a new TEITOK corpus meant for learner corpora. This learner corpus provides a number of learner-oriented settings, that can be modified if necessary:

  • The CSS style provides a manuscript oriented view
  • The CSS style also provides a speaker-turn oriented view, with the @who of each utterance in front of each transcribed utterance
  • Each token is provided with multiple levels of target hypotheses with their annotations
  • The teiHeader edit is set up for a learner data and task data

As the owner of this project you may wish to at least do the following things, and do the first two as soon as possible:

  • Login as nobody@nowhere.com / defaultpassword
  • Add yourself as an admin user in user edit, and remove the dummy admin user.
  • Run additional checks on your configuration in config check.
  • Replace the main.tpl template to reflect your name, and ideally even giving it your own look-and-feel (colours, logos, etc.).
  • Refined your settings.
  • Replace this page by a page explaining your project.

More information about how to set-up TEITOK can be found in the help pages: http://www.teitok.org/index.php?action=help
