Skeleton Site

This is an empty site to help set-up a new TEITOK corpus page

As the owner of this project you may wish to at least do the following things, and do the very first asap:

  • Login as nobody@nowhere.com / defaultpassword
  • Add yourself as an admin user in user edit, and remove the dummy admin user.
  • Run additional checks on your configuration in config check.
  • Replace the main.tpl template to reflect your name, and ideally even giving it your own look-and-feel (colours, logos, etc.).
  • Choose which attributes to use on tokens in your settings.xml.
  • Replace this page in page edit by a page explaining your project.

More information about how to set-up TEITOK can be found in the help pages: http://alfclul.clul.ul.pt/teitok/site/index.php?action=help
